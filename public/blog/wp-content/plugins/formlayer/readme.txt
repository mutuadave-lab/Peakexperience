Build fast and powerful contact forms in WordPress with an intuitive drag-and-drop builder packed with smart features.

== Description ==

FormLayer is a state-of-the-art form builder that leverages the power of modern web technologies to provide a stunning, intuitive drag-and-drop experience. Whether you need a simple contact form or a complex multi-step application, FormLayer provides the tools to build it in minutes.

Built with performance and aesthetics in mind, FormLayer features an administrative interface, real-time live previews, and a sophisticated design system that makes form building a joy rather than a chore.

=== Key Free Features ===
* **Intuitive Drag & Drop Builder**: Build complex layouts with ease.
* **15+ Standard Fields**: Name, Email, Text, Dropdown, Radio, Checkboxes, and more.
* **Real-time Preview**: See exactly how your form looks on Desktop, Tablet, and Mobile.
* **Modern Templates**: Start with pre-built templates for common use cases.
* **Entry Management**: View and manage all submissions directly in your WordPress dashboard.
* **Responsive by Default**: All forms look great on every device out of the box.
* **AJAX Submissions**: No page reloads for a smoother user experience.
* **Spam Protection**: Built-in support for hCaptcha (free), Cloudflare Turnstile and Google reCAPTCHA (Pro).

=== Premium Pro Features ===
* **Advanced Fields**: Phone, Ratings, Password, File Uploads, and more.
* **Professional Templates**: Access to a library of high-conversion templates.
* **Powerful Integrations**: Connect to Mailchimp, Slack, Discord, Google Sheets, Notion, and Trello.
* **Custom Styling**: Advanced control over colors, fonts, and spacing.
* **Priority Support**: Get help when you need it most.
* **Multi-column Layouts**: Create professional grid-based forms effortlessly.

== Installation ==

1. Upload the `FormLayer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to FormLayer in your admin menu to start building forms.

== Frequently Asked Questions ==

= Is FormLayer responsive? =
Yes, all forms created with FormLayer are 100% responsive and optimized for mobile devices.

= Can I export my entries? =
Yes, you can export your form submissions to CSV directly from the entries dashboard.
